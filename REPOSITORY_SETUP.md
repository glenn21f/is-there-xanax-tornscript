# GitHub repository setup

Use these values on the GitHub "Create a new repository" screen.

## General

**Repository name**

```text
is-there-xanax-tornscript
```

**Description**

```text
DroqsDB-first Torn userscript that checks Xanax stock in Japan, UK, and South Africa, predicts landing availability, and falls back to YATA/cache.
```

## Configuration

**Visibility**

```text
Public
```

A private repo will make auto-updates annoying because userscript managers need a public raw file URL.

**Add README**

```text
Off
```

Upload the included `README.md` instead.

**Add .gitignore**

```text
No .gitignore
```

Upload the included `.gitignore` instead.

**Add license**

```text
No license
```

Upload the included `LICENSE` instead, or choose **MIT License** directly on GitHub if you prefer.
